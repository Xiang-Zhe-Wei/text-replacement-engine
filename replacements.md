# 口語用字與修正對照表（3.14.2025 修訂）

此檔案定義口語用字與修正用字的對應關係，供文字替換工具使用。
如需新增替換規則，請依照相同格式在下方新增一行。

## 格式說明

```
| No | 口語用字 | 修正用字 |
```

> 標註 `*(需依文意選擇)*` 的條目含多個候選詞或不定長度樣式（如 `有...問題`），工具**不會**自動替換，須手動處理。
